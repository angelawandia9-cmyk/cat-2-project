# Zuri Café Website

A fully responsive multi-page website for **Zuri Café**, a fictional café based in Westlands, Nairobi, Kenya. Built as a Web Application Development assignment using HTML5, CSS3, JavaScript ES6+, and Bootstrap 5.

## Pages
- **Home** — Hero banner and highlights
- **About** — Our story, mission, values, and team
- **Menu** — Full food and drinks menu with prices in KSh
- **Gallery** — Filterable photo gallery (Coffee / Food / Events)
- **Blog** — News and stories from the café
- **Testimonials** — Customer reviews and star ratings
- **Contact** — Contact form with JavaScript validation
- **FAQ** — Frequently asked questions with Bootstrap accordion

## JavaScript Features
- **Gallery Filter** — Filter photos by category with instant show/hide
- **Form Validation** — Validates name, email, and message before submission

## Technologies
- HTML5, CSS3, JavaScript ES6+
- Bootstrap 5.3
- Git & GitHub
